## Запуск

1. Клонировать репозиторий:
   ```bash
   git clone https://github.com/FROSTYBYTHEWAY/Fundamentals-of-algorithmization.git
   ```
2. Открыть нужный `.cs`-файл в Visual Studio
3. Запустить через `F5` или `Ctrl+F5`
